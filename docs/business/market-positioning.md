# Market positioning

Visual HTML je prakticky AI nastroj na premenu screenshotov, UI obrazoviek a dokumentov na cisty HTML/CSS. Produkt ma cielit na pouzivatelov, ktori nechcu iba inspiracny kod, ale realny vystup, ktory mozu otvorit, skontrolovat, upravit a pouzit.

## Positioning statement

Visual HTML turns screenshots and document images into editable, standards-based HTML/CSS with live preview, refinement, export, and quality checks.

## What the product is

- Screenshot-to-HTML converter.
- OCR + visual reconstruction workflow.
- AI assistant pre frontend reprodukciu.
- HTML export tool pre UI a dokumenty.
- QA-friendly generator s warnings a refinementom.

## What the product is not

- Nie je iba landing-page generator.
- Nie je Figma plugin ako primarny produkt.
- Nie je iba OCR data extractor.
- Nie je iba chat prompt wrapper.
- Nie je finalny no-code website builder.

## Product narrative

Manual screenshot recreation is slow. Generic AI chat can help, but the workflow is messy: upload an image, write a prompt, copy code, open a local file, fix broken CSS, ask for another revision, and repeat.

Visual HTML packages that workflow into one product: upload, generate, preview, refine, export, and compare.

## Brand tone

- Practical.
- Precise.
- Developer-friendly.
- Honest about uncertainty.
- Focused on usable output.

Avoid overpromising pixel-perfect magic. Stronger claim: faithful, editable reconstruction with transparent warnings and fast iteration.
